# ECPI University Coursework

This directory contains documentation and summaries of coursework completed at ECPI University, showcasing foundational and advanced skills in IT administration, networking, cybersecurity, and cloud solutions. Each subdirectory represents a distinct course, detailing its objectives, learning outcomes, and key topics covered.

*   [CIS101: Computer Configuration I](./cis101-computer-configuration-i/README.md)

*   [CIS142: Introduction to Cloud Solutions](./cis142-introduction-to-cloud-solutions/README.md)
*   [CIS150: Introduction to Networking](./cis150-introduction-to-networking/README.md)
