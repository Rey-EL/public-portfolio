# Windows Active Directory (CIS256)

## Course Overview
In this course, I gained extensive hands-on experience with the application and utilization of Windows Active Directory components. I mastered the skills necessary to manage, monitor, and optimize desktop and user environments, conducting thorough analyses of current and planned business models. My learning included determining current and future expansion processes, alongside the implementation and application of common security processes within the Windows environment. This course equipped me with the expertise to design and maintain robust Active Directory infrastructures.

## Key Learning Outcomes
- **Active Directory Installation:** Successfully performed an installation of Active Directory on a Windows member server, establishing foundational directory services.
- **Active Directory Object Planning:** Developed comprehensive plans for creating and organizing Active Directory objects, ensuring logical and efficient directory structures.
- **Group Policy Strategy:** Administered and implemented a strategic plan for managing Group Policy within Active Directory, enforcing security and configuration settings across the network.
- **Active Directory Infrastructure Configuration:** Analyzed and configured the processes required to establish and maintain a resilient Active Directory Infrastructure.
- **Backup and Restore Strategy:** Designed and planned a robust Backup and Restore Strategy for Active Directory, ensuring data integrity and business continuity in the event of a disaster.
