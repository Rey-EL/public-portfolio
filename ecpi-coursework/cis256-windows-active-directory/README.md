# Windows Active Directory (CIS256)

## Course Overview
This course will provide students with hands-on application and use of windows active directory components. Students will learn to manage, monitor, and optimize desktop and user environments, analyzing current and planned business models, determining current and future expansion processes, as well as the implementation and use of common security processes in the windows environment.

## Key Learning Outcomes
- Perform an installation of Active Directory on a Windows member server.
- Create a plan for creating Active Directory objects in Active Directory.
- Administer a plan to implement and manage a Group Policy Strategy for Active Directory.
- Analyze the processes to configure an Active Directory Infrastructure.
- Plan a Backup and Restore Strategy for Active Directory.

## Technologies and Tools
- Active Directory
- Certificate Services
- DHCP
- DNS
- Group Policy
- PowerShell
- VCASTLE
- Windows Server 2019
