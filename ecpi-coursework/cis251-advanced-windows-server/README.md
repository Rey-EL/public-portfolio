# CIS251: Advanced Windows Server

## Course Description
This course provides students with the knowledge and skills necessary to install, manage, monitor, configure, and troubleshoot Windows Server. Topics include DNS, DHCP, Remote Access, Network Protocols, and IP Routing in a Windows network infrastructure. Students will also learn about Network Address Translation and Certificate Services.
