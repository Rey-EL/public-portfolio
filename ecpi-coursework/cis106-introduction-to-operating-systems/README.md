# Introduction to Operating Systems (CIS106)

## Course Overview
This course provides an introduction to the major hardware and software components of computer-based operating systems. Students will learn about Windows and Linux. They will also learn basic system maintenance, use of terminal commands, data security, virtualization, and computer numbering systems. Upon successful course completion, students will be able to configure the user interface, perform basic maintenance, and conduct data backup and recovery.

## Key Learning Outcomes
- Compare and contrast various types of hardware including RAM, hard disk, processors, and graphics cards and software including operating system, application, virtualization, and utility software.
- Use basic operating system and disk utilities to perform system configuration and maintenance tasks.
- Discuss the history of operating systems.
- Apply command line interface in various operating systems.
- Examine security and ethical use of data, computer systems, mobile, and other devices.
- Identify the different numbering systems used in computers, tablets, and devices such as Decimal, Binary, and Hexadecimal.
