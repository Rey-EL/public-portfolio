# CIS150: Introduction to Networking

## Course Description
This course provides an introduction to the basic concepts, technology, and terminology used in computer networks. Students will learn to diagram a network, configure network devices, and connect them in order to access the internet. The course also covers basic security and essential network troubleshooting. By the end of this course, students will be able to design and implement a small network with associated security controls.

## Course Learning Outcomes
Upon completion of this course, students will be able to:
1. Define common networking terms.
2. Identify elements that could be in a home network.
3. Describe the benefits of documenting a network.
4. Describe the role of packets in network communication.
5. Explain the binary numbering system and how it is used in networking.
6. Identify the parts of an IPv4 address.
7. Configure IPv4 addresses.
8. Describe common networking topologies.
9. Describe network protocols and how they are layered into a network.
10. Describe the OSI and TCP/IP networking models.
11. Describe cable characteristics of the major types of cables used in computer networking.
12. Create an Ethernet cable.
13. Determine the best placement of a home network router.
14. Explain the fundamentals of Packet Tracer.

## Key Topics Covered
*   Introduction to Networking (Basic concepts, IP addresses, IoT devices)
*   Binary Numbering System
*   Network Hardware and Models (Cables, Wi-Fi, Routers, Switches)
*   OSI and TCP/IP Models
*   Packet Tracer Fundamentals
