# CIS142: Introduction to Cloud Solutions

## Course Description
This course will introduce cloud computing architecture and security concepts. Students will learn about the benefits of cloud computing, cloud characteristics, cloud models and solutions along with deployment methods. Students will also gain knowledge of hardware, storage, thin clients and virtualization in the cloud and skills to implement cloud security fundamentals with virtualization security management. Upon successful course completion, students will be able to apply current cloud computing technologies and environments.

## Course Learning Outcomes
Upon completion of this course, students will be able to:
1. Evaluate the concepts, characteristics, delivery models and benefits of cloud computing environments, virtual networks, and necessary resources.
2. Evaluate the characteristics, use cases, and choices available for Public, Private, and Hybrid Cloud Computing.
3. Create, manage, and monitor compute and storage resources.
4. Evaluate the key technical, organization, security, and compliance cloud computing challenges.
5. Implement business continuity and disaster recovery strategies.

## Key Topics Covered
*   Introduction to Cloud Computing (Virtualization, Hypervisors, Containers)
*   Cloud Compute Services (Scaling, Security Posture, Compliance)
*   Cloud Storage Services (Cloud-based vs. On-premise, Providers)
*   Cloud Security Services (Data Protection, Access Controls, Threat Management)
*   Cloud BCDR Services (Disaster Recovery, Backup, Failover)
