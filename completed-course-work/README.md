# Completed Course Work

This directory contains completed exercises, worksheets, and smaller assignments from various training programs. These items are distinct from the full-scale projects found in the main portfolio section.

### Contents

-   **[Completed Access Control Worksheet](./completed-access-control-worksheet.md)**
    *   This worksheet demonstrates the ability to analyze an access control scenario, identify authorization issues, and recommend appropriate technical, operational, and managerial controls. It showcases an understanding of concepts like Role-Based Access Control (RBAC) and the principle of least privilege.

-   **[Completed Data Leak Worksheet](./completed-data-leak-worksheet.md)**
    *   This worksheet demonstrates the ability to analyze a data leak incident, identify the contributing factors, and recommend improvements based on the principle of least privilege (NIST SP 800-53: AC-6). It showcases an understanding of how to apply the NIST Cybersecurity Framework (CSF) to protect against data leaks.

-   **[Incident Handler's Journal](./incident-handler-journal.md)**
    *   This journal entry documents a ransomware incident at a healthcare clinic. It demonstrates the ability to apply the "5 W's" (Who, What, When, Where, Why) to document and analyze a security event, a foundational practice in incident response.
